# 부하테스트 결과 파일 저장 위치

이 폴더에는 다양한 GPU 모델의 부하테스트 결과 파일(이미지 또는 텍스트)을 저장합니다.

## 📁 파일 위치

```
public/benchmark-results/
```

## 📄 지원하는 파일 형식

### 이미지 파일
- PNG (`.png`)
- JPG/JPEG (`.jpg`, `.jpeg`)
- 기타 이미지 형식

### 텍스트 파일
- TXT (`.txt`)

## 💡 사용 방법

1. **파일 저장**: 부하테스트 결과 파일을 이 폴더(`public/benchmark-results/`)에 저장하세요.
   - 파일명은 자유롭게 지정할 수 있습니다. LLM이 파일 내용을 분석하여 GPU 모델, 워크로드 타입, 문서 타입 등을 자동으로 추출합니다.

2. **애플리케이션 재시작**: 파일을 저장한 후 애플리케이션을 재시작하면 자동으로 로드됩니다.

3. **UI에서 사용**: 입력 페이지에서 "개발자가 제공한 문서 사용" 체크박스를 선택하면 저장된 파일이 자동으로 사용됩니다.

## 📋 파일 내용 형식

### 텍스트 파일 예시
```
GPU 모델: H100
워크로드 타입: OCR
문서 타입: 진료비 영수증

컨테이너 수 | RPS | Latency (ms)
1          | 0.58| 712
5          | 1.82| 173
10         | 2.45| 245
```

### 이미지 파일
- 부하테스트 결과가 포함된 스크린샷 또는 PDF를 이미지로 변환한 파일
- 테이블 형식의 데이터가 포함된 이미지

## ⚠️ 주의사항

- 파일명에 특별한 규칙이 없어도 됩니다. LLM이 파일 내용을 분석하여 필요한 정보를 자동으로 추출합니다.
- 파일 내용에 GPU 모델, 워크로드 타입, 성능 데이터 등이 포함되어 있으면 자동으로 인식됩니다.
